Backman 支持所有 Bootstrap3 的样式与功能，但是却又与 Bootstrap 有所不同，比如许多元素默认就被带上了样式，因此很多地方不再需要记忆繁多的 Bootstrap 样式名与页面结构，从而降低了新手的上手条件也提高了平均开发效率
